# 计算机视觉实验三：图像缩小、恢复与频域分析
## 实验题目
- 使用OpenCV读入一幅灰度图像，先对图像进行下采样，再分别用不同内插方法恢复图像尺寸，并结合傅里叶变换和DCT变换对原图与恢复图进行分析比较
## 实验内容与要求
- **任务1**：图像读入与预处理
- **任务2**：下采样
- **任务3**：显示原图，用 Matplotlib 或 OpenCV 显示原图
- **任务4**：转换为灰度图，将彩色图转换为灰度图，并显示
- **任务5**：保存处理结果，把灰度图保存为新文件
- **任务6**：用 NumPy 做一个简单操作，例如输出某个像素值，或者把图像左上角一块区域裁剪出来保存
- **任务7**：如果需要多个文件进行编译，如何实现
## 运行环境
- 操作系统：Ubuntu Linux
- VS Code C++
## 基本信息
- test.jpg：原图
- cropped.jpg：裁剪图片
- gray_test.jpg：灰度图
- main.cpp：源文件
