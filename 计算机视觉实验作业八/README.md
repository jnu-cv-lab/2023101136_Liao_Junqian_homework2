## 实验内容与要求
### 任务1：环境准备与图像数据集加载
- 完成深度学习实验环境配置，并加载图像数据集，完成以下内容：
1. 安装并测试 PyTorch、torchvision、NumPy、Matplotlib 等依赖库；
2. 成功导入 PyTorch 并判断当前环境是否支持 GPU 加速；
3. 完成简单 PyTorch 张量操作验证环境可用性；
4. 加载 MNIST 与 CIFAR-10 图像数据集，查看数据集基本信息；
5. 显示若干张样本图像及其真实类别标签。
### 任务2：数据集划分
- 将图像数据集按比例完成划分：
1. 将原始训练集按 8:2 划分为新训练集与验证集；
2. 保留官方测试集用于最终模型评估；
3. 说明训练集、验证集、测试集分别用于模型训练、效果验证与最终性能评估。
### 任务3：CNN 模型结构定义
- 构建适配图像分类的卷积神经网络模型，完成以下内容：
1. 设计包含卷积层、激活函数、池化层、全连接层、输出层的完整 CNN 结构；
2. 保证模型输入尺寸匹配 MNIST/CIFAR-10 图像大小，输出类别数与数据集一致；
3. 在报告中清晰说明每层网络的参数、功能与维度变化。
### 任务4：CNN 模型训练
- 使用训练集完成卷积神经网络训练，完成以下内容：
1. 选择交叉熵作为损失函数，选取 Adam 优化器进行参数更新；
2. 至少训练 10 个 epoch，完成前向传播、损失计算、反向传播与参数优化；
3. 每个 epoch 记录训练损失 training loss 与训练准确率 training accuracy。
### 任务5：模型验证与效果评估
- 使用验证集监测模型训练过程，完成以下内容：
1. 每个 epoch 训练完成后在验证集上测试模型性能；
2. 记录验证损失 validation loss 与验证准确率 validation accuracy；
3. 对比训练集与验证集的损失、准确率差异，判断模型是否过拟合。
### 任务6：模型测试与结果展示
- 使用独立测试集评估模型最终性能，完成以下内容：
1. 计算模型在测试集上的损失与分类准确率；
2. 显示若干张测试图像，同时标注真实标签与模型预测标签；
3. 整理并输出模型最终测试结果。
### 任务7：训练曲线绘制
- 可视化模型训练全过程，完成以下内容：
1. 以 epoch 为横轴，分别绘制训练损失与验证损失曲线；
2. 以 epoch 为横轴，分别绘制训练准确率与验证准确率曲线；
3. 为图表添加标题、坐标轴标签与图例，保证可视化清晰规范。
### 任务8：实验结果综合分析
- 基于训练曲线与测试结果完成全面分析，回答以下问题：
1. 训练损失是否随 epoch 增加逐步下降；
2. 验证准确率是否随训练过程逐渐提升；
3. 训练准确率与验证准确率是否存在明显差距；
4. 若存在差距，分析过拟合产生的可能原因；
5. 统计并分析数据集中最容易被错误分类的类别；
6. 对比 MNIST 与 CIFAR-10 任务难度，并说明原因。
### 进阶任务1：CNN 网络结构改进
- 修改基础卷积神经网络结构并对比性能，完成以下内容：
1. 增加卷积层数，调整卷积核数量，修改全连接层神经元数量；
2. 在网络中加入 Dropout 层防止过拟合；
3. 说明具体修改内容，对比修改前后测试准确率变化；
4. 分析准确率提升或下降的核心原因。
### 进阶任务2：不同优化器效果对比
- 比较多种优化器对训练效果的影响，完成以下内容：
1. 选用 Adam 与 SGD 两种优化器完成对照实验；
2. 记录不同优化器、学习率与对应测试准确率；
3. 简要分析优化器原理差异对收敛速度、最终精度的影响。
### 进阶任务3：MNIST 与 CIFAR-10 数据集对比
- 完成两大数据集全流程实验并横向对比，完成以下内容：
1. 对比两个数据集的图像类型、类别数量、测试准确率；
2. 评估并说明两个数据集的分类难度差异；
3. 详细解释 CIFAR-10 图像分类任务难度更高的核心原因。
