# 传统机器学习方法在手写数字图像分类中的应用
## 图像信息
- digits_samples：数据集样本图像
- error_samples：错误分类样本
- confusion_matrix：混淆矩阵（热力图形式）
## 实验结果与分析
- 见计算机视觉实验报告七
## 实验内容与要求
### 任务1：数据准备
- 加载手写数字图像数据集，并完成以下内容：
1. 查看数据集中图像的数量；
2. 查看每张图像的大小；
3. 查看类别标签；
4. 显示若干张样本图像及其真实标签。
### 任务2：数据划分
- 将数据集划分为：训练集；测试集。要求测试集比例为 25% 左右。
- 说明训练集和测试集分别用于什么目的。
### 任务3：特征表示
- 将每张图像转换为机器学习模型可以处理的特征向量。
- 要求说明：
1. 一张 8 × 8 图像如何变成 64 维向量；
2. 为什么传统机器学习方法通常需要这种特征转换；
3. 原始像素作为特征有什么优点和局限。
### 任务4：模型训练
- 请至少使用以下传统机器学习方法中的五种完成分类实验：
1. 最近邻方法，KNN；
2. 朴素贝叶斯分类器；
3. 逻辑回归；
4. 支持向量机，SVM；
5. 决策树；
6. 随机森林。
- 每种方法都需要完成：
1. 训练模型；
2. 在测试集上进行预测；
3. 记录分类准确率。
### 任务5：结果比较
- 将不同模型的测试准确率整理成表格。
- 要求根据实验结果回答：
1. 哪个模型准确率最高？
2. 哪个模型准确率最低？
3. 不同模型之间的表现差异是否明显？
4. 你认为产生差异的原因是什么？
### 任务6：错误样本分析
- 选择一个表现较好的模型，分析其分类错误情况。
- 要求：
1. 绘制或展示混淆矩阵；
2. 找出若干个被错误分类的样本；
3. 分析哪些数字最容易被混淆；
4. 说明这些错误样本为什么可能被模型误判。
### 思考题
- 请在实验报告最后回答以下问题：
- 为什么传统机器学习方法需要先把图像转换为特征向量？
- KNN、SVM、决策树和随机森林的分类思想有什么不同？
- 为什么单棵决策树容易过拟合？
- 为什么随机森林通常比单棵决策树更稳定？
- 如果图像发生平移、旋转或缩放，直接使用原始像素作为特征会有什么问题？
