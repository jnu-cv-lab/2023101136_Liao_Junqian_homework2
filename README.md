# 计算机视觉实验二：c++图像基本读写

## 实验任务
- **任务1**：使用 OpenCV 读取一张测试图片
- **任务2**：输出图像基本信息，在终端打印：图像尺寸、图像通道数、像素数据类型（例如：宽度、长度）
- **任务3**：显示原图，用 Matplotlib 或 OpenCV 显示原图
- **任务4**：转换为灰度图，将彩色图转换为灰度图，并显示
- **任务5**：保存处理结果，把灰度图保存为新文件
- **任务6**：用 NumPy 做一个简单操作，例如输出某个像素值，或者把图像左上角一块区域裁剪出来保存
- **任务7**：如果需要多个文件进行编译，如何实现
## 运行环境
- 操作系统：Ubuntu Linux
- VS Code C++
## 基本信息
- test.jpg：原图
- cropped.jpg：裁剪图片
- gray_test.jpg：灰度图
- main.cpp：源文件

## 实验二学习报告
- 如果需要多个文件进行编译：
- 把所有 .cpp 都写进 tasks.json 的 args 里，如：
- "args": [
-    "-g",
-    "main.cpp",        // 主文件
-    "project1.cpp",           // 文件1
-    "project2.cpp",           // 文件2
-    "-o",
-   ...
- ]
