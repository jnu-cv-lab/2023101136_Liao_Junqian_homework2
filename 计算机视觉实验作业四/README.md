# 计算机视觉实验四
## 实验内容与要求
- **第一部分**
- 生成棋盘格和chirp测试图
- 直接下采样，观察混叠
- 加高斯滤波再下采样，对比差异
- 画 FFT 频谱，在频域上确认混叠消失了
- **第二部分**：验证σ公式
- 固定 M=4，分别用 σ=0.5, 1.0, 2.0, 4.0 做实验
- 观察 σ 太小（混叠残留）和 σ 太大（过度模糊）的效果
- 找到 "最合适" 的 σ，和理论值 σ≈0.45M=1.8 对比
- **第三部分**：自适应下采样
- 用梯度分析估计局部 M 值
- 对不同区域用不同 σ 滤波
- 和全图统一下采样对比误差图
- **思考题**
- 如果对一张人脸照片做4倍下采样，人脸区域和背景区域应该用相同的σ吗？为什么？如果不同，怎么决定各自用多大的σ？
